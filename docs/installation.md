# Installation

## Stable release

To install traincheck, run this command in your terminal:

```sh
uv add traincheck
```

Or if you prefer to use `pip`:

```sh
pip install traincheck
```

## From source

The source files for traincheck can be downloaded from the [Github repo](https://github.com/darkleia/traincheck).

You can either clone the public repository:

```sh
git clone https://github.com/darkleia/traincheck
```

Or download the [tarball](https://github.com/darkleia/traincheck/tarball/main):

```sh
curl -OJL https://github.com/darkleia/traincheck/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd traincheck
uv sync
```
